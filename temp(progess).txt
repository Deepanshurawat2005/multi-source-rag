So far, we have successfully built the core PDF RAG pipeline. The project can load a PDF, split it into chunks, generate embeddings using nomic-embed-text, store the embeddings in FAISS, retrieve relevant chunks, and generate answers using llama3.1. We have also created and integrated three external tools: Wikipedia Search, Arxiv Search, and Web Search (DuckDuckGo). Additionally, we built a fallback agent that can use these tools to answer questions. At this point, the PDF RAG system, external tools, and fallback agent are all working independently.
